MicroCommerce API (E-Ticaret Backend Sistemi)

Teknolojiler: .NET 8, PostgreSQL, JWT, MediatR, FluentValidation, EF Core

Kurumsal düzeyde Clean Architecture prensipleri uygulanarak; Domain, Application, Infrastructure ve WebAPI katmanları ile yüksek test edilebilirliğe sahip bir backend altyapısı geliştirildi.

CQRS deseni ile iş mantığı ve veri erişimi ayrıştırıldı; ASP.NET Core Identity ve JWT entegrasyonu ile güvenli kimlik doğrulama (Bearer Token) ve yetkilendirme mekanizmaları kuruldu.
