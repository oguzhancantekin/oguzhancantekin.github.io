MicroCommerce API (Kurumsal E-Ticaret Backend)

Teknolojiler: .NET 8, C#, PostgreSQL, JWT, MediatR, FluentValidation, Entity Framework Core

Kurumsal düzeyde Clean Architecture prensipleri tam olarak uygulanarak; Domain, Application, Infrastructure ve WebAPI katmanları ile yüksek test edilebilirliğe sahip, güçlü bir backend altyapısıdır. CQRS deseni ile iş mantığı ve veri erişimi ayrıştırılmış; ASP.NET Core Identity ve JWT entegrasyonu ile (Bearer Token) güvenli yetkilendirme inşa edilmiştir.

Scan Me! / Ürün Tarama (Mobil Uygulama)

Teknolojiler: Flutter, Dart, SQFlite

Kullanıcıların tüketim ürünlerinin içeriklerini kamera aracılığıyla tarayarak kişiselleştirilmiş alerjen tespiti yapmasını sağlayan, fayda odaklı mobil uygulama geliştirmesi. İnternet erişimi olmadan da verimli çalışabilmesi için SQFlite yerel veritabanı mimarisi kurgulanmıştır.

Tedarikçi Yönetim Paneli (Web Uygulaması)

Teknolojiler: C#, ASP.NET MVC, Entity Framework, MSSQL, Bootstrap

Uyumsoft stajı kapsamında, tedarikçilerin sisteme giriş yaparak ürün/sipariş yönetimini kolaylıkla gerçekleştirebildiği web tabanlı yönetim platformudur. MVC mimarisiyle modüler olarak tasarlanmış olup; rol tabanlı erişim (Admin vb.), CRUD standartları, yetkilendirme (Auth) ve e-posta ile bildirim gibi entegre modüller içerir.
